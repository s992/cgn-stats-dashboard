All commands are listed with their ! equivalents, but can also be used prefixed with / (in chat) or sm_ (in console). For example, you can use !r, /r, or sm_r interchangeably.

Map Commands

!maplist
Displays all maps categorized by their difficulty.
!mapperlist
Displays the map authors and the number of maps they have made. In the sub-menu it will list the name of each map.
!mi / !mapinfo

Displays information for a map. This information includes:

  • Author
  • Type (staged/linear)
  • Number of stages/checkpoints
  • Number of bonuses
  • Submenu to view point breakdown
    • Number of completions
    • Points for map completion
    • Points per stage
    • Total points for stages+map
  • Point Brackets: Based off of total number of completions. These numbers will constantly be changing.
  • Points for stage record
  • Submenu displaying points per top 10 time
    • Displays points for rank 1-10, these are based off of map tier. Group points for rank 1-x are added on to these numbers.
!nominate
Opens a menu where you can nominate a map for the next vote.
!nextmap
Displays the next map.
!points
Displays the point breakdown for a map. Usage: !points surf_jusched_beta1 (name has to be exact)
!revote
Allows you to change your vote during the vote for next map.
!rtv
Vote to skip the current map
!ve
Vote to extend the current map

Timer Commands

!cpr
Compares your personal record to the world record. Alternatively, you can compare someone else's personal record to the world record. Usage: !cpr, !cpr @3 (compares rank 3 to world record), !cpr Flaudgin
!cp
Displays a person's location/progress on a map.
!gb / !goback
Sends you back to the previous stage.
!hud
Toggles HUD visibility.
!hudconfig

Allows you to customize your HUD. CS:GO has an engine limitation of three lines.

Arrange every character in the order you'd like information to be displayed on the HUD.

  • a = Current Time
  • A = Current Time (Reduced)
  • b = Current Activity
  • B = Current Activity (Reduced)
  • c = Current Zone (Reduced)
  • C = Current Zone
  • d = Current Map
  • e = Speed (X,Y)
  • E = Speed (X,Y,Z)
  • f = Active WR
  • g = Active PR
  • i = Spectators
  • j = Show Checkpoint
  • J = Checkpoint on new line
  • k = Reduced Speed (X,Y)
  • K = Reduced Speed (X,Y,Z)
  • l = Active WR(no stage)
  • m = Active PR(no stage)
  • n = Current Zone (Extended)
  • | = Creates | character
  • # = Creates new line

Example: !hudconfig Aj|i#C|f#k|g will give you the default hud.

Example HUD layouts:

aJ#E#f

a|b#e|f#dc

Aj|i#C|f#k|g

!lookup
Search for a player's profile when the player isn't currently in the server.
!p

Displays profile information for a player. Contains the following information:

  • Rank x/xxx
  • Total Points
  • Group Points
  • Completion Percentage
  • Maps: x/xxx (points earned for map completions)
  • Stages: x/xxx (points earned for stage completions)
  • Bonuses: x/xxx (points earned for bonus completions)
  • Top Times: xxx (points earned for top 10s)
  • Records
  • Maps: xxx (points for map WRs)
  • Stages: xxx (points for stage WRs)
  • Bonus: xxx (points for bonus WRs)
  • Player Info
    • Steam ID
    • First time connecting
    • Last time connecting
    • Region
    • Steam community profile link
    • Back to player profile
!pr
Allows you to check the personal record information for yourself or another player. Usage: !pr, !pr Flaudgin
!r
Sends you to the start of the map.
!report

Allows you to report:

  • Bugged, glitched, or superboosted times
  • Maps that are broken (have a jail, or something else that needs to be removed or fixed)
  • Maps that have bonuses that aren't zoned
!rr
Displays recently broken world records.
!rrs
Displays recently broken stages records.
!rrb
Displays recently broken bonus records.
!st / !surftop
Displays the current top 50 surfers.
!sl / !tl
Allows you to save your current location and velocity in order to practice or learn a map. When you use !sl it will give you a number. With !tl you can teleport to your most recent save, or you can use !tl 4 (or any other number) to use any previous saves or saves from another player.
!teleport
Sends you to the start of the stage you are currently on.
!timer
Toggles your timer. Disabling your timer allows the use of !tl, !fly, and !goto.
!top / !wr
Displays the top times for a map.
!ts / !topstage
Displays the top times for a specific stage on a map. Usage: !ts 3 (or any other number), !ts 2 surf_timewarp (so you can check for maps that aren't currently being played)
!tb / !topbonus
Displays the top times for a specific bonus on a map. Usage: !tb 2 (or any other number), !tb 1 surf_nyze (so you can check for maps that aren't currently being played)
!options
Configure timer options.

Misc Commands

!donate
Brings you to the donation page.
!fly
Allows you to noclip through the map.
!goto
Allows you to teleport to a specific player.
!hide
Allows you to hide other player models. This option is saved on the server.
!ranks
Displays the rank titles and their requirements.
!knife / !guns
Allows you to change or hide your knife/weapon. This option is saved on the server.
!myid
Displays your ID number, based off of the first time you connected to the server.
!psay
Allows you to private message an admin.
!showkeys
Displays keys pressed for the player you are spectating.
!showmykeys
Displays the keys you are pressing.
!spec
Sends you to spectator.
!specinfo
Allows you to see who is spectating you or another player.
!hidechat
Toggles chat visibility.
!sm / !selfmute
Mutes a player. Usage: !sm Renegade
!su / !selfunmute
Unmutes a player. Usage: !su Renegade
!cm / !checkmute
Displays a list of muted players.
!hop / !servers
Displays a list of available servers and allows you to join the server.
